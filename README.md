# SistemaAcademico
Sistema para analisar o desempenho acadêmico dos alunos
Baseado em tecnologias como Next.JS e tailwindcss para o front end e Express com autenticação JWT para o backend.
Tem como principal objetivo auxiliar os usuários a entender melhor seu desempenho acadêmico. O sistema mostra um relatório com gráficos detalhados sobre o seu desempenho, e consegue gerar um arquivo com as principais informações do relatório
